
# คู่มือการใช้งาน Calculator-tlaohachai

สร้างไว้ใช้ บวก ลบ คูณ หาร

สร้างตอนเรียน Node.js

- สามารถบวกเลขได้
- ลบเลขได้
- คูณเลขได้

## วิธีใช้งาน

ต้องลง NodeJS ก่อนนะ [ดาวน์โหลดได้ที่นี่](https://nodejs.org/en/)

วิธี require module

```js
const calculator = require('calculator-tlaohachai');
```
